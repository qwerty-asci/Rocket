Descripción detallada

Implementación del entorno de cohete para entrenamiento con DQN.

Este archivo define la clase Rocket que simula un cohete controlable mediante acciones discretas. Se implementa la integración numérica mediante el método Runge-Kutta de cuarto orden. La clase está expuesta a Python usando pybind11.
