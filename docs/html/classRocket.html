Descripción detallada

Clase que define el entorno del cohete.

El cohete puede recibir acciones discretas:

  • 0: No hacer nada
  • 1: Encender/apagar motor
  • 2: Rotar boquilla a la derecha
  • 3: Rotar boquilla a la izquierda

Proporciona funciones para:

  • reset(): Reiniciar el entorno
  • step(action): Avanzar un paso de simulación
  • sample(): Obtener una acción aleatoria
